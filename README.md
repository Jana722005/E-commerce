# Musalamma Grocery Shop

A full-stack modern grocery e-commerce web application built using the MERN stack with responsive UI, Razorpay payment integration, admin dashboard, wishlist, basket system, order management, and authentication.

---

# Features

## User Features

- User Registration & Login
- JWT Authentication
- Responsive Dashboard
- Product Search & Filtering
- Add to Basket
- Wishlist System
- Quantity Management
- Checkout System
- Razorpay Payment Gateway
- Cash on Delivery
- User Orders Page
- Profile Management
- Notifications System

---

## Admin Features

- Admin Dashboard
- Add Products
- Manage Products
- Update Products
- Delete Products
- Manage Orders
- Update Order Status
- Real-time Order Monitoring
